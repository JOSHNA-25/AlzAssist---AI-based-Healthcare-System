# AlzAssist - AI based-Healthcare-System
AlzAssist is an artificial intelligence (AI) based healthcare assistive web application designed to assist Alzheimer's patients to perform daily activities in an independent and safe manner. The system supports technologies like face recognition, voice recognition, speech processing, reminder automation and AI-generated responses, delivering intelligent healthcare support. The use of the webcam for facial recognition allows patients to locate family members and familiar faces, whereas the implementation of the Web Speech API supports patients' communication with ease using speech-to-text and text-to-speech features. The Groq API enables the creation of intelligent responses and the provision of reminders, guidance and day-to-day help to patients. The reminder management module alerts users of medications, meals, appointments and other activities via a visual reminder and a voice reminder. The application is developed using React.js, HTML, CSS, JavaScript and Vite to provide a simple, responsive and user-friendly interface for elderly users and Alzheimer’s patients.
